# Database

## Описание проекта

"Примитивная база данных" — приложение, написанное на Python, позволяющее создавать базу данных и делать с ней различные операции."

## Установка

Для установки зависимостей используйте команду:
``make install``
Для запуска игры используйте команду:
``make project``
